
# Daily Text Insights

A small Python project for extracting simple insights from text files.

## Features

- Word count
- Most common words
- Basic text preprocessing (lowercasing, punctuation removal, stopword filtering)

This project is intentionally minimal and designed as a lightweight introduction to text analysis and basic NLP concepts.

## How to run

1. Make sure you have Python 3 installed.
2. Place your text inside `sample.txt`.
3. Run the script from the project directory:

```bash
python text_insights.py
